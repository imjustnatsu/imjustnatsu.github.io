Aide pour OS

Les bases

  • "mkdir x" permet de créer un dossier
  • "cat x" permet de voir/lire un fichier
    exemple :
    cat /etc/group
    cat /etc/passwd
    cat /etc/shadow
  • "passwd x" permet de changer le mot d'un utilisateur (x)
  • Faire attention a bien être dans le root et pas dans le home
